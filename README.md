# Soulithra Nexus Game

## 🎮 Description
Soulithra Nexus is a browser game where the player controls a character, collects coins and progresses through levels. Built on HTML, CSS and JavaScript.

## 🚀 Functionality
- Points system and timer
- Character movement with arrows
- Coins to collect
- Automatic levels
- Basic design and styling
